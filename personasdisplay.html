personas and storyboarding

context:
- for this project i analyzed user interactions and experiences with a toaster oven interface
- after observing and interviewing users, i created two personas based on the users that described their thoughts, feelings, words, and actions while using the interface
- i then illustrated a storyboard displaying one of my personas interacting with the interface

lessons learned:
- observing and interviewing users is an important method to identify usability issues with an interface
- designers should have users interact with a prototype before creating their final designs
- in general, users tended to be frustrated by the overcomplicated interface & knobs without explanations/instructions
